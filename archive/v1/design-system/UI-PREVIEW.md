# UI 重设计预览

我已为你制定了完整的 UI 重设计方案！以下是核心改进：

## 🎯 核心问题 & 解决方案

### 问题 1: 功能展示不完整
**现状**: Panopticon 监控页面独立存在，主仪表盘无法看到关键监控数据
**解决**: 将核心监控指标集成到主仪表盘

### 问题 2: 信息密度低
**现状**: Agent 卡片过于简单，一屏只能看 1-2 个
**解决**: 重新设计紧凑卡片，一屏展示全部 6 个 Agent + 监控图表

### 问题 3: 缺少科技感
**现状**: 黑白设计略显单调
**解决**: 深色主题 + 青色强调色，类似 NASA 指挥中心风格
